 ## Instructions on running project locally:

Clone the project

```bash
  git clone https://github.com/Hrishikesh332/RetroNexus.git
```

Install dependencies:

```bash
    cd www.retronexus.in
    npm install
```

Run Project on terminal

```bash
  npm run dev
```
